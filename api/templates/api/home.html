Geo Melt

Geo Melt is an API that allow users to extract and transform data

coming from xml sources such Kml from Google Maps or OSC from Open Street Maps

We can extract data from geojson files as well.

The web site and the API is now under contruction, so we are looking forward to be functional very soon.
